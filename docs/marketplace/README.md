# 숨고·크몽 등록 준비 산출물

이 폴더는 업로드 전 최종 확인을 위한 작업물 모음입니다. 실제 플랫폼 업로드는 사용자 확인 후 진행합니다.

## 파일

- `pricing-research.md`: 경쟁가 조사 요약, 정상가/5월 이벤트가 후보 밴드, 최종 가격 확정 규칙
- `service-registration-copy.md`: 7개 서비스별 크몽·숨고 등록 원고
- `competitor-benchmark.md`: 상위 판매자 등록 구조 벤치마킹과 AIO 적용 원칙
- `kmong-upgrade-copy.md`: 크몽 전용 고도화 상세 원고, 추가 가격, FAQ, 요청사항
- `profile-company-info.md`: 전문가 프로필, 기업정보, 응답 템플릿
- `final-review-checklist.md`: 업로드 전 검수 체크리스트
- `asset-manifest.md`: 생성된 대표 이미지와 상세 이미지 목록

## 이미지

업로드 후보 이미지는 `public/marketplace`에 생성되어 있습니다.

- 크몽 대표 이미지: `*-kmong-cover.png`
- 숨고 대표 이미지: `*-soomgo-cover.png`
- 상세 검수 이미지: `*-detail-proof.png`
- 프로필 이미지: `aio-profile-square.png`
- 크몽 고도화 상세 이미지: `public/marketplace/kmong-upgrade/*-kmong-detail-*.png`
