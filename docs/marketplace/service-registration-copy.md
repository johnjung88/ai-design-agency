# 숨고·크몽 등록용 상세페이지 원고

공통 원칙:

- AI 제작 언급 없음.
- 외부 연락처, 이메일, 사이트 URL, SNS, 카카오톡 문구 없음.
- 가격 문구는 본문이 아니라 플랫폼 가격 옵션 영역에만 입력.
- `최저가`, `1위`, `100% 만족`, `매출 보장`, `환불 보장`, `무제한` 문구 사용 금지.
- 수정 범위는 계약 범위 내 단순 수정으로 제한한다.

## 1. 웹사이트 제작

### 상품명 후보

- 크몽: 브랜드 신뢰를 높이는 반응형 웹사이트를 맞춤 제작해 드립니다
- 숨고: 브랜드와 문의 전환을 고려한 반응형 웹사이트 제작

### 서비스 요약

회사 소개, 서비스 안내, 포트폴리오, 문의 동선까지 한 번에 정리되는 반응형 웹사이트를 제작합니다. 단순히 화면을 만드는 것이 아니라 방문자가 어떤 정보를 보고 문의까지 이동해야 하는지 구조부터 정리합니다.

### 추천 대상

- 처음 홈페이지가 필요한 소상공인·개인사업자
- 기존 사이트가 있지만 문의 전환이 약한 브랜드
- 서비스 소개, 예약, 문의, 포트폴리오가 한 화면 흐름으로 필요한 팀
- 빠른 일정 안에 검토 가능한 결과물이 필요한 초기 사업자

### 제공 범위

- PC·모바일 반응형 화면 구성
- 메인 메시지와 섹션 구조 정리
- 문의 폼, 지도, CTA 등 기본 전환 동선
- 기본 SEO 메타 정보 정리
- 배포 또는 운영 가이드 제공

### 패키지별 차이

- STANDARD: 랜딩 1P
- DELUXE: 홈피 5P
- PREMIUM: 홈피 10P

### 작업 과정

1. 목적, 업종, 참고 사이트, 필요한 페이지 확인
2. 메뉴 구조와 섹션 흐름 제안
3. PC·모바일 화면 제작
4. 1차 결과 공유 및 수정 반영
5. 최종 파일, 배포 정보 또는 운영 가이드 전달

### 납품물

- 반응형 웹사이트 결과물
- PC·모바일 화면 확인 자료
- 배포 또는 운영 가이드
- 요청 범위에 따른 이미지·문구 교체 안내

### 수정 범위

- 포함: 오탈자, 문구 교체, 이미지 교체, 버튼/섹션 내 단순 배치 수정
- 별도 협의: 페이지 추가, 신규 기능 추가, 전체 콘셉트 변경, 복잡한 외부 시스템 연동

### 준비 자료

- 사업자/브랜드 소개
- 서비스 설명
- 로고와 브랜드 컬러
- 참고 사이트 2~3개
- 넣고 싶은 이미지와 필수 문구

### FAQ

Q. 자료가 정리되어 있지 않아도 가능한가요?  
A. 가능합니다. 상담 단계에서 필요한 페이지와 우선순위를 먼저 정리합니다.

Q. 모바일 화면도 포함되나요?  
A. 모든 패키지는 모바일 확인을 포함합니다. 범위는 패키지별 페이지 수에 따라 달라집니다.

Q. 작업 후 직접 수정할 수 있나요?  
A. 운영 방식에 따라 수정 가능한 구조와 가이드를 안내합니다.

## 2. 쇼핑몰·카페24 디자인

### 상품명 후보

- 크몽: 브랜드몰 첫인상을 살리는 쇼핑몰 메인·카페24 디자인을 제작합니다
- 숨고: 쇼핑몰 메인 화면과 상품 진열 흐름을 정리해 드립니다

### 서비스 요약

카페24, 브랜드몰, 커머스 사이트의 첫 화면과 상품 진열 흐름을 정리합니다. 업종에 맞는 메인 비주얼, 배너, 카테고리, 베스트 상품 영역을 구성해 고객이 상품을 빠르게 이해하도록 돕습니다.

### 추천 대상

- 쇼핑몰 오픈 전 메인 화면 구성이 필요한 브랜드
- 기존 쇼핑몰의 첫인상이 약한 운영자
- 배너와 상품 진열이 정리되지 않아 전환 흐름이 끊기는 브랜드
- 카페24 기반으로 관리 가능한 디자인 방향이 필요한 사업자

### 제공 범위

- 쇼핑몰 메인 구성 방향
- 배너/상품 진열 섹션 디자인
- 모바일 쇼핑 흐름 기준 화면 정리
- 카테고리와 베스트 상품 영역 제안
- 운영자가 교체하기 쉬운 이미지 기준 안내

### 패키지별 차이

- STANDARD: 쇼핑몰 메인
- DELUXE: 쇼핑몰 풀세팅
- PREMIUM: 쇼핑몰 풀커스텀

### 작업 과정

1. 업종, 상품군, 참고몰, 브랜드 톤 확인
2. 메인 화면 구조와 핵심 배너 방향 정리
3. PC·모바일 기준 디자인 제작
4. 피드백 반영
5. 최종 이미지와 운영 가이드 전달

### 납품물

- 메인 화면 이미지
- 배너/섹션 이미지
- 모바일 확인 이미지
- 이미지 교체 가이드

### 수정 범위

- 포함: 문구 교체, 상품 이미지 교체, 색상 일부 조정, 배너 문구 수정
- 별도 협의: 상품 촬영, 전체 리브랜딩, 결제/회원 기능 개발, 외부 솔루션 고도화

### 준비 자료

- 쇼핑몰 URL 또는 오픈 예정 플랫폼
- 브랜드 로고와 컬러
- 대표 상품 이미지
- 상품 카테고리
- 참고 쇼핑몰 2~3개

### FAQ

Q. 카페24가 아니어도 가능한가요?  
A. 가능합니다. 다만 실제 적용 방식은 플랫폼에 따라 달라질 수 있습니다.

Q. 상품 등록까지 포함되나요?  
A. 기본은 디자인과 화면 구성입니다. 상품 등록 대행은 별도 옵션으로 협의합니다.

## 3. 로고·명함

### 상품명 후보

- 크몽: 바로 사용할 수 있는 로고·명함·프로필 키트를 제작합니다
- 숨고: 브랜드 첫인상을 위한 로고와 명함 디자인

### 서비스 요약

작은 브랜드도 바로 사용할 수 있도록 로고, 심볼, 명함, 온라인 프로필 이미지를 함께 정리합니다. 인쇄와 온라인 노출을 모두 고려해 확장 가능한 형태로 제작합니다.

### 추천 대상

- 신규 브랜드를 시작하는 소상공인
- 로고와 명함을 한 번에 정리하고 싶은 대표님
- SNS, 프로필, 견적서에 통일된 이미지를 쓰고 싶은 브랜드
- 기존 로고의 톤을 정리하고 싶은 사업자

### 제공 범위

- 메인 로고 또는 심볼
- 가로형/세로형 적용 방향
- 명함 양면 디자인
- PNG/SVG/PDF 등 용도별 파일
- 온라인 프로필 이미지

### 패키지별 차이

- STANDARD: 로고 단독
- DELUXE: 로고+명함
- PREMIUM: 풀브랜딩

### 작업 과정

1. 브랜드명, 업종, 원하는 분위기 확인
2. 콘셉트 방향 정리
3. 로고 시안 제작
4. 선택안 수정 및 명함 적용
5. 최종 파일 전달

### 납품물

- 로고 이미지 파일
- 명함 인쇄용 파일
- 온라인 사용 이미지
- 컬러와 사용 방향 안내

### 수정 범위

- 포함: 색상, 폰트 톤, 간격, 문구, 명함 정보 수정
- 별도 협의: 브랜드명 변경, 완전한 콘셉트 변경, 추가 시안 제작, 인쇄 대행

### 준비 자료

- 브랜드명
- 업종과 타깃 고객
- 원하는 분위기 키워드
- 명함에 넣을 정보
- 참고 이미지 2~3개

### FAQ

Q. 로고만 먼저 진행할 수 있나요?  
A. 가능합니다. 명함과 프로필 이미지는 패키지에 따라 추가됩니다.

Q. 인쇄까지 포함되나요?  
A. 기본은 디자인 파일 납품입니다. 인쇄는 별도 협의가 필요합니다.

## 4. 상세페이지

제품의 특징, 사용 장면, 구매 이유를 세로형 상세페이지로 정리합니다. 단순히 예쁜 이미지가 아니라 고객이 제품을 이해하고 선택할 수 있도록 메시지와 흐름을 구성합니다.

- 스마트스토어, 오픈마켓, 카페24 상세페이지가 필요한 판매자
- 제품 설명은 있지만 디자인 흐름이 부족한 브랜드
- 모바일에서 읽기 쉬운 판매 페이지가 필요한 사업자
- 기존 상세페이지를 개선하고 싶은 운영자

### 제공 범위

- 상세페이지 구조 설계
- 제품 핵심 카피 정리
- 모바일 기준 세로형 디자인
- 이미지 보정과 섹션 구성
- 웹용 JPG/PNG 납품

### 패키지별 차이

- STANDARD: 상세 1장
- DELUXE: 상세 5장 세트
- PREMIUM: 상세 풀스택

### 작업 과정

1. 제품 정보와 타깃 고객 확인
2. 핵심 소구 포인트 정리
3. 세로형 흐름 구성
4. 디자인 제작 및 수정 반영
5. 최종 이미지 전달

### 납품물

- 상세페이지 JPG/PNG
- 모바일 확인 이미지
- 섹션별 카피 정리본
- 플랫폼 업로드용 이미지

### 준비 자료

- 제품명과 주요 특징
- 판매 타깃
- 상세 스펙
- 참고 상세페이지 2~3개

Q. 스마트스토어에 바로 올릴 수 있나요?  
A. 플랫폼 규격에 맞는 이미지 형태로 전달합니다.

## 5. PPT 디자인

### 상품명 후보

- 크몽: 제안서·회사소개서·발표자료 PPT를 맞춤 디자인해 드립니다
- 숨고: 발표 목적에 맞춘 PPT 디자인과 구성 정리

### 서비스 요약

회사소개서, 제안서, IR, 강의자료, 발표자료를 목적에 맞게 정리합니다. 정보 구조, 가독성, 도식화, 톤앤매너를 맞춰 발표와 제출에 바로 사용할 수 있는 PPT 파일로 납품합니다.

### 추천 대상

- 회사소개서나 제안서가 필요한 사업자
- 기존 PPT를 더 깔끔하게 바꾸고 싶은 팀
- 발표 흐름과 시각 구조가 필요한 창업자
- 강의·세미나 자료를 전문적으로 보이게 만들고 싶은 분

### 제공 범위

- PPT 톤앤매너 정리
- 표지, 목차, 본문 슬라이드 디자인
- 도식화, 아이콘, 그래프 정리
- PPTX와 PDF 납품
- 기본 발표 흐름 제안

### 패키지별 차이

- STANDARD: PPT 10P
- DELUXE: PPT 20P
- PREMIUM: PPT 30P+

### 작업 과정

1. 발표 목적과 대상 확인
2. 원고와 기존 자료 검토
3. 1~2장 기준 톤 제안
4. 전체 슬라이드 디자인
5. 수정 반영 후 PPTX/PDF 전달

### 납품물

- PPTX 원본
- PDF 파일
- 사용 이미지/폰트 안내
- 슬라이드별 간단 메모

### 수정 범위

- 포함: 오탈자, 문구 교체, 색상 일부 조정, 이미지 교체, 도식 일부 수정
- 별도 협의: 원고 전체 재작성, 자료 조사, 대량 장수 추가, 전체 콘셉트 변경

### 준비 자료

- 기존 PPT 또는 원고
- 발표 목적과 대상
- 원하는 장수
- 브랜드 로고와 컬러
- 참고 스타일

### FAQ

Q. 원고만 있어도 가능한가요?

Q. 원본 파일을 받을 수 있나요?  
A. PPTX 원본과 PDF를 함께 전달합니다.

## 6. 자동화·앱

### 상품명 후보

- 크몽: 반복 업무 자동화와 MVP 대시보드를 맞춤 제작해 드립니다
- 숨고: 업무 자동화·데이터 처리·MVP 화면 제작

### 서비스 요약

반복 업무, 데이터 정리, 운영 대시보드, MVP 화면을 실제 업무 흐름에 맞춰 제작합니다. 필요한 기능을 작게 정의하고, 사용자가 바로 확인할 수 있는 결과물 중심으로 진행합니다.

### 추천 대상

- 엑셀·CSV 반복 작업을 줄이고 싶은 사업자
- 예약 발행, 수집, 정리 같은 업무 흐름을 자동화하고 싶은 팀
- MVP나 내부 운영 화면이 필요한 초기 사업자
- 외부 API나 DB 연동이 필요한 운영자

### 제공 범위

- 요구사항 정리
- 데이터 입력/출력 구조 설계
- 자동화 스크립트 또는 화면 제작
- 테스트 데이터 검증

## 7. 영상 콘텐츠

### 상품명 후보

- 크몽: 브랜드 홍보 영상과 숏폼 콘텐츠를 목적에 맞게 제작합니다
- 숨고: 숏폼·홍보 영상·브랜드 인트로 제작

### 서비스 요약

브랜드 인트로, 서비스 홍보 영상, 쇼츠·릴스용 세로 영상을 제작합니다. 짧은 시간 안에 브랜드 메시지와 행동 유도 문구가 보이도록 구성합니다.

### 추천 대상

- 짧은 브랜드 홍보 영상이 필요한 사업자
- 인스타그램, 유튜브 쇼츠, 릴스용 영상이 필요한 브랜드
- 서비스 소개를 영상으로 정리하고 싶은 팀
- 기존 이미지와 화면 자료를 영상으로 바꾸고 싶은 분

### 제공 범위

- 영상 구성안
- 자막과 장면 흐름
- BGM과 기본 효과
- 16:9 또는 9:16 영상 제작
- 플랫폼용 출력 파일

### 패키지별 차이

- STANDARD: 30초 숏폼 1개
- DELUXE: 60초 홍보 영상 또는 숏폼 3개
- PREMIUM: 브랜드 영상 세트, 구성안 포함

### 작업 과정

1. 영상 목적과 플랫폼 확인
2. 사용할 이미지, 화면, 문구 정리
3. 구성안 작성
4. 영상 편집 및 자막 적용
5. 수정 반영 후 최종 파일 전달

### 납품물

- MP4 영상 파일
- 자막 적용본
- 썸네일 이미지
- 플랫폼별 비율 안내

### 수정 범위

- 포함: 오탈자, 자막 교체, 이미지/클립 일부 교체, BGM 조정
- 별도 협의: 촬영, 성우 녹음, 전체 스토리 변경, 추가 비율 제작

### 준비 자료

- 영상 목적
- 사용할 사진·영상·로고
- 넣을 문구
- 참고 영상 2~3개
- 원하는 플랫폼과 비율
