# 숨고·크몽 상위 판매자 벤치마킹

조사 기준일: 2026-04-30

## 핵심 결론

상위 판매자는 가격을 가장 앞에 세우기보다 `구체적인 결과물`, `명확한 작업 범위`, `빠른 응답`, `후기/포트폴리오 신뢰`, `수정 기준`을 먼저 보여준다. AIO는 2026년 5월 이벤트가를 클릭 진입 장치로 쓰고, 상세 설명에서는 정상가 대비 할인보다 `빠른 납품이 가능한 정리형 제작 시스템`과 `기획 포함 맞춤 제작`을 강조한다.

## 공통 등록 패턴

| 항목 | 상위 판매자 패턴 | AIO 적용 방식 |
| --- | --- | --- |
| 상품명 | 구매자가 검색하는 결과물 키워드를 앞에 둠 | `웹사이트 제작`, `쇼핑몰 세팅`, `PPT 디자인`, `상세페이지 제작`처럼 카테고리 키워드 선명화 |
| 대표 이미지 | 서비스 종류와 완성 분위기를 즉시 인지시키는 이미지 | 가격·연락처 없이 서비스명, 패키지 범위, 결과물 샘플 중심 |
| 패키지 | STANDARD는 진입용, DELUXE는 주력, PREMIUM은 고급 앵커 | 이벤트가는 판매가에 입력, 정상가는 패키지 설명/상담 안내에서 보조 표기 |
| 설명 구조 | 추천 대상, 제공 범위, 작업 과정, 납품물, 수정 기준을 반복 | 7개 서비스 모두 동일한 구매 판단 흐름 유지 |
| 신뢰 요소 | 리뷰, 거래 수, 응답 시간, 포트폴리오, 원본/상업 이용 가능 여부 | 승인 전에는 포트폴리오 이미지와 사업자/세금계산서 가능 정보로 보완 |
| 운영 방식 | 결제 전 문의 유도, 준비 자료를 명확히 요구 | 범위 확인 후 착수, 신규 기능/페이지/기획 변경은 별도 협의 |

## 크몽 벤치마킹

- 좋은 상품명은 감성 문장보다 구매 키워드가 강하다. 30자 제한을 감안해 `맞춤형 웹사이트 제작`, `PPT 디자인 제작`처럼 짧고 직접적인 제목을 우선한다.
- 크몽의 3단 패키지는 가격 차이보다 `작업 범위 차이`가 명확해야 한다. 페이지 수, 시안 수, 수정 횟수, 작업일, 원본 제공 여부를 같은 언어로 맞춘다.
- PPT 상위 상품은 `기획/리디자인/구성+디자인`처럼 난이도 기준을 분리하고, 원본 파일·상업적 이용·폰트/이미지 라이선스 안내를 상세 설명에 둔다.
- 상세 이미지는 단순 배너보다 실제 결과물 느낌을 보여주는 긴 이미지가 유리하다. 디자인/PPT 계열은 여러 장의 상세 이미지가 심사와 구매 판단에 모두 유리하다.
- 리뷰가 적은 신규 판매자는 `빠른 납품`, `명확한 수정 범위`, `사업자 정보`, `포트폴리오 사용 범위`를 자세히 써서 불확실성을 줄여야 한다.

## 숨고 벤치마킹

- 숨고는 고객이 여러 고수의 프로필과 견적을 비교하는 구조이므로, 마켓 상품보다 `고수 프로필`, `포트폴리오`, `응답 템플릿`의 신뢰도가 중요하다.
- 숨고 마켓 설명에는 가격, 연락처, 외부 홍보문구가 들어가면 승인 리스크가 있다. 상품 설명은 가격표보다 제공 범위와 과정 중심으로 작성한다.
- 2026년 5월 30일부터 마켓 신규 등록·수정·판매가 중단되므로, 마켓은 5월 이벤트용으로만 활용하고 장기 수주는 고수 프로필과 견적 응답 흐름으로 받는다.
- 숨고 고객은 가격만 보는 경우가 많아 보이지만, 실제 선택 단계에서는 포트폴리오 확인과 일정/수정/사후 안내가 중요하다. 낮은 가격 경쟁보다 “짧은 범위의 빠른 제작”으로 설명한다.

## AIO 상품명 보정안

크몽 제목은 짧고 검색형으로, 숨고 상품명은 조금 더 설명형으로 운영한다.

| 서비스 | 크몽 상품명 | 숨고 상품명 |
| --- | --- | --- |
| 웹사이트 | 맞춤형 웹사이트 제작 | 반응형 웹사이트 제작 |
| 쇼핑몰 | 쇼핑몰 세팅 제작 | 쇼핑몰 메인·풀세팅 제작 |
| 로고·명함 | 로고와 명함 디자인 제작 | 로고·명함 브랜드 키트 제작 |
| 상세페이지 | 상세페이지 디자인 제작 | 구매 흐름을 정리한 상세페이지 제작 |
| PPT | PPT 디자인 제작 | 제안서·회사소개서 PPT 디자인 |
| 자동화·앱 | 업무 자동화와 앱 MVP 제작 | 업무 자동화·앱 MVP 제작 |
| 영상 콘텐츠 | 숏폼·브랜드 영상 제작 | 숏폼·브랜드 영상 콘텐츠 제작 |

## 패키지 운영 원칙

- STANDARD: 빠른 결정을 유도하는 최소 범위 상품. 작업 범위가 작고 수정 횟수는 1회 중심.
- DELUXE: 실제 주문을 가장 많이 받도록 설계한 주력 상품. AIO의 “기획 포함 맞춤 제작”을 가장 잘 보여준다.
- PREMIUM: 고급 범위와 앵커 가격 역할. 추가 상담이 필요한 경우가 많으므로 “범위 확인 후 진행” 문구를 넣는다.
- 모든 상품에 `작업일`, `수정 횟수`, `납품물`, `준비 자료`, `추가 비용 기준`을 고정 항목으로 넣는다.
- 이벤트 문구는 `2026년 5월 오픈 이벤트가`로만 제한한다. `최저가`, `1위`, `무제한`, `100% 만족`, `매출 보장`, `환불 보장`은 사용하지 않는다.

## 남은 등록 전 수정 체크

- 쇼핑몰, 로고·명함, 상세페이지, 자동화·앱, 영상 콘텐츠의 제목을 위 보정안 기준으로 짧게 정리한다.
- 대표 이미지에 가격, 연락처, URL, SNS, 카카오톡 문구가 없는지 다시 확인한다.
- 크몽 패키지 설명에는 이벤트가와 정상가를 분리하되, 본문 첫 화면은 작업 범위 중심으로 둔다.
- 숨고 설명에는 가격표를 넣지 않고 옵션명과 제공 범위로 분리한다.
- 자동화·앱은 기능 범위가 크게 달라질 수 있으므로 `상담 후 기능 단위 확정` 문구를 필수로 넣는다.
- 영상 콘텐츠는 원본 소스, 자막, 음원/이미지 라이선스, 수정 범위를 명확히 적는다.
