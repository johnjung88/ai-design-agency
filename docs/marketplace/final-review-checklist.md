# 업로드 전 최종 검수 체크리스트

## 가격

- [ ] 모든 서비스에 `정상 판매가`와 `2026년 5월 오픈 이벤트가`가 분리되어 있다.
- [ ] 플랫폼 판매가는 `2026년 5월 오픈 이벤트가` 기준으로 입력한다.
- [ ] 이벤트 기간은 `2026년 5월 접수 건 한정`으로만 표현한다.
- [ ] 본문 이미지와 상세 설명에 직접적인 할인율, 가격 비교, 과장 문구를 넣지 않았다.
- [ ] 크몽/숨고 업로드 직전 경쟁 상품 5개 이상을 다시 확인했다.

## 문구

- [ ] AI 제작 언급이 없다.
- [ ] 외부 연락처, 이메일, 사이트 URL, SNS, 카카오톡 문구가 없다.
- [ ] `최저가`, `1위`, `유일`, `무제한`, `100% 만족`, `매출 보장`, `환불 보장` 문구가 없다.
- [ ] 작업 범위, 작업일, 수정 횟수, 납품물이 명확하다.
- [ ] 수정 범위와 별도 협의 항목이 분리되어 있다.
- [ ] 숨고 상품 설명에는 가격 문구가 없다.

## 이미지

- [ ] 크몽 대표 이미지는 4:3 비율이며 텍스트가 중앙 안전 영역에 있다.
- [ ] 숨고 대표 이미지에는 가격과 연락처가 없다.
- [ ] 서비스별 상세 이미지에 포트폴리오 예시가 3개 이상 들어 있다.
- [ ] 프로필 이미지는 1:1 비율이다.
- [ ] 민감 정보가 포함된 화면은 공개 전에 다시 확인한다.

## 플랫폼 입력

- [ ] 크몽은 7개 서비스를 각각 별도 카테고리에 등록한다.
- [ ] 크몽 상위 카테고리는 최초 제출 후 변경이 어렵기 때문에 제출 전 다시 확인한다.
- [ ] 숨고는 마켓 7개 등록과 고수 프로필/포트폴리오 세팅을 함께 진행한다.
- [ ] 숨고 마켓 종료 일정 때문에 5월 이벤트용으로만 운영하고, 장기 수주는 고수 프로필·견적 수주 동선에 둔다.

## 사용자 최종 확인 항목

- [ ] 가격표
- [ ] 대표 이미지
- [ ] 상세페이지 문구
- [ ] 전문가/기업정보
- [ ] 포트폴리오 공개 범위
- [ ] 업로드 순서
